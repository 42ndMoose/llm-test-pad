LLM-Friendly Starter (Minimal)
================================

Files:
- index.html — minimal, server-rendered test page with explicit end marker.
- robots.txt — allows all crawlers.
- llm.json — JSON mirror of core content.

Quick deploy options:
1) Netlify Drop: https://app.netlify.com/drop (drag & drop this folder or the zip)
2) GitHub Pages: create a repo, add these files at root, enable Pages.
3) Cloudflare Pages / Vercel: connect a repo and deploy as a static site.

After deployment, share the public URL so an LLM browser can test extraction and citation.
